# Universe Auctions API

## Description

This service provides API for managing and accessing data related to Universe auctions

## Requirements

- NodeJS version 14+
- NPM

## Required External Service

- MongoDB

## Primary Third Party Libraries

- NestJS
- Mongoose (MongoDB)

## Installation

```bash
npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger

To check API doc, use the similar link below:
<http://localhost:8080/v1/doc>
